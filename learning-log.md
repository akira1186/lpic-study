###### lspciコマンドが見つからないエラー

\###状況

`lspci`コマンドを実行しようとしたが、インストールされていないため実行できなかった。

\###解決方法

`pciutils`パッケージをインストールすることで解決した。

sudo apt install pciutils



###### lsusbコマンドが見つからないエラー

\###状況

`lsusb`コマンドを実行しようとしたが、インストールされていないため実行できなかった。

\###解決方法

`usbutils`パッケージをインストールすることで解決したが、何も変化がなく、それは環境が仮想的なものでUSB機器が一つも繋がっていない状態だからであると分かった。
